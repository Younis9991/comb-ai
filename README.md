## ✨ What is comb-ai?

comb-ai helps you work with AI tools by managing context better. It reduces the number of AI tokens you use, which can save time and cost. It uses a structured memory bank that remembers important details. This way, you can access your information faster with less repetition.

The tool organizes your AI workflow by storing useful data and reading it smartly. It works behind the scenes to optimize how your AI reads documents and handles conversations. You don’t need to know coding to use it.

### Download or install fails  

- Make sure you downloaded the whole file without interruption  
- Try downloading from a different browser or internet connection  
- Run the installer as Administrator (right-click > Run as Administrator)  

### Memory bank is empty after adding files  

- Verify files are supported types (text, PDF, markdown)  
- Wait a moment for the app to process the files fully  
- Restart the app if the problem persists
